Tipo Caracter

Tópicos de Ajuda >> Informaçãos de Uso >> Dados e Variaveis >> Tipo Caracter

O que é?

  Qualquer digito número ou alfanumérico ou letra que pode ser inserido por um dispositivo de entrada.

Como usar?

  O tipo caracter deve estar acompanhado de aspas simples. Veja na tabela como utilizar:

TIPO CARACTER
Formas de Utilizar Exemplo Descrição
1 caracter sexo tipo caracter com a variavel sexo
2 caracter consoante, vogal tipo caracter para usuario atribuir uma letra nas variaveis

Veja o exemplo completo:

Exemplo 1:

  programa
  {
    funcao inicio( )
    {
      caracter sexo

      sexo = 'f'
    }
  }

Exemplo 2:

  programa
  {
       funcao inicio( )
       {
          caracter letra

          escreva ("Digite uma letra\n")
          leia (letra)

          escreva (letra,"\n")
       }
  }
