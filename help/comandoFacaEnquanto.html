Comando faca-enquanto

Tópicos de Ajuda >> Informações de Uso >> Comandos Básicos >> Comando faca-enquanto

O que é?

  O comando FACA-ENQUANTO é semelhante ao comando ENQUANTO, permite que uma sequencia de instruções sejam executadas, porém o teste ocorre no final do laço, ou seja a instrução será executada pelo menos uma vez.

Como usar?

  Para utilizar o comando FACA-ENQUANTO você deve iniciar com a palavra reservada FACA, abrir e fechar chaves e entre elas adicionar as instruções a serem executadas, e no fim do laço deve ser aplicado entre parênteses a condição a ser testada. Veja na tabela como utilizar.

COMANDO FACA-ENQUANTO

    faca
  {
     <instrução>
  } enquanto (<condição>)

Veja os exemplos:

Exemplo 1:

  programa
  {
      funcao inicio ( )
      {
         inteiro x

         x = 1
         faca
         {
            x = x + 1
            escreva (x)
         } enquanto (x<=5)
      }
  }

Exemplo 2:

  programa
  {
        funcao inicio( )
        {
            inteiro cont = 0

            faca
            {
               se (cont % 2 != 0)
               {
                           escreva ("É número ímpar ", cont, "\n")
               }
               cont++
            }enquanto (cont < 30)
        }
  }
