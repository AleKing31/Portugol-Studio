O que é um algoritmo?

Tópicos de Ajuda >> Informações de Uso >> O que é um algoritmo?

Existem diversos conceitos para a palavra algoritmos. Alguns estão relacionados à resolução de problemas num âmbito geral, tais como:
• Algoritmo é uma seqüência de passos finita para solução de um determinado problema.
• Algoritmo é um conjunto de ações ordenadas que levam à realização de uma tarefa.

Outras definições consideram a visão da Ciência da Computação:
• Algoritmo é uma seqüência de passos que visa atingir um objetivo bem
definido (FORBELLONE, 1999). • Algoritmo é uma seqüência finita de instruções ou operações cuja execução,
em tempo finito, resolve um problema computacional (SALVETTI, 1999). • Algoritmo é a descrição lógica de um programa de computador que pode-
rá ser codificado, posteriormente, em linguagens de programação.
