O que é?

  O comando nova linha \n serve para ocorrer o salto próxima linha, .

Como usar?

  Sempre deve estar entre aspas duplas. Veja no exemplo como utilizar:

Exemplo:

  programa
  {
      funcao inicio( )
      {
         inteiro a, b, pares = 0

         escreva ("Digite dois numero inteiros \n")
         leia (a,b)
         enquanto (a<=b)
         {
           se(a % 2 == 0)
           {
            pares = pares + 1
           }
          a = a + 1
         }
         escreva ("A quantidade de números pares: ", pares, "\n")
      }
  }
