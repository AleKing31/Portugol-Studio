O que é?

  Em algoritmos as operações relacionais servem para realizar comparações que terão como resultado um valor lógico.

Operadores Relacionais

  > (maior)
  < (menor)
  > = (maior igual)
  < = (menor igual)
  = = (igual)
  != (diferente)
