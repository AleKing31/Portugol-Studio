Função

Tópicos de Ajuda >> Informações de Uso >> Comandos Básicos >> Função

O que é?

  Uma função é semelhante ao procedimento, porém tem uma diferença a função retorna um valor o procedimento não retorna valor.

Como usar?

  Veja na tabela a sintaxe e como usar:

FUNçãO
  funcao <tipo> <nome_da_funcao>( )
  {
     <instruções>
  }
  funcao <tipo> <nome_da_funcao>(<tipo> <variavel>)
  {
     <instrução>
  }

Veja os exemplos:

Exemplo:

  programa
  {
      funcao inicio( )
      {
         inteiro n

         escreva ("Digite um número \n")
         leia (n)
         escreva ("A soma dos números é ", somatorio(n))
      }

      funcao inteiro somatorio(inteiro n)
      {
         inteiro soma = 0

         para(inteiro i=1; i<=n; i++)
         {
            soma = soma + i
         }
          retorne (soma)
      }
  }

Exemplo 2:

  programa
  {
        funcao inicio( )
        {
             inteiro num1, num2
             escreva ("Digite dois Numeros \n")
             leia (num1, num2)

              se (igualdade(num1,num2)) {
               escreva ("\nOs dois numeros sao iguais")
                } senao {
                escreva ("\nOs dois numeros sao diferentes")
        }

        funcao logico igualdade (inteiro num1, inteiro num2)
        {
             se (num1 == num2)
             {
               retorne (verdadeiro)
             }senao{
               retorne (falso)
             }
        }
  }
