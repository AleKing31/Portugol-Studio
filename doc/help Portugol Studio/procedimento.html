O que é?

  Procedimentos são subrotinas que tem por objetivo executar alguma operação especifica, não retornando valor algum.

Como usar?

  Veja na tabela a sintaxe e como usar:

PROCEDIMENTO
  funcao <tipo> <nome_da_funcao>( )
  {
     <instruções>
  }
  funcao <tipo> <nome_da_funcao>(<tipo> <variavel>)
  {
     <instrução>
  }

Veja o exemplo:

Exemplo 1:

  programa
  {
       funcao inicio( )
       {
         inteiro valor

         escreva ("Digite valor\n")
         leia (valor)
         frase (valor)
         escreva (" Tchau! \n")
       }

       funcao frase(inteiro valor)
       {
         inteiro cont

         para(cont=0; cont<=valor; cont++)
         {
            escreva ("NUNCA DESISTA \n")
         }
       }
  }
