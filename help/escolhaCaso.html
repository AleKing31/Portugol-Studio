Escolha Caso

Tópicos de Ajuda >> Informações de Uso >> Comandos Básicos >> escolha-caso

O que é?

  O comando ESCOLHA CASO é semelhante ao switch case da linguagem PHP. Determina qual caso será executado dependendo da escolha do usuário.

Como usar?

  Veja na sintaxe e no exemplo como utilizar o escolha caso.

COMANDO ESCOLHA CASO

    escolha(<expressão> ou <variavel>)
  {
    caso 1:
       <instrução>
    pare
    caso 2:
       <instrução>
    pare
    caso 3:
       <instrução>
    pare
    caso contrario:
       <instrução>
  }

Veja o exemplo

Exemplo:

  programa
  {
      funcao inicio( )
      {

         inteiro esc, voto1 = 0, voto2 = 0, voto3 = 0

         escreva ("Escolha um dos candidatos abaixo, digitando o número do candidato!! \n")
         escreva ("1->Dilma Rousseff \n")
         escreva ("2->José Serra \n")
         escreva ("3->Marina Silva \n")
         leia (esc)

          enquanto (esc != 0)
          {
              escolha (esc)
              {
                caso 1:
                voto1=voto1 + 1
                pare
                caso 2:
                voto2=voto2 + 1
                pare
                caso 3:
               voto3=voto3 + 1
                pare
                caso contrario
                escreva ("\n Este não é um numero de um dos candidatos!\n ")
              }
             leia (esc)
          }
          escreva ("\n O candidato 1 tem ", voto1)
          escreva ("\n O candidato 2 tem ", voto2)
          escreva ("\n O candidato 3 tem ", voto3)
      }
  }
