limpa

Tópicos de Ajuda >> Informaçãos de Uso >> Comandos Básicos >> limpa

O que é?

  O comando "limpa( )" limpa a tela do programa quando ele estiver sendo executado.

Como usar?

  limpa( ). Veja no exemplo.

Exemplo:

  programa
  {
     funcao inicio( )
     {
        inteiro num

        escreva ("Digite um número")
        leia (num)

        limpa ( )
        escreva ("O número é ", num)
     }
 }
