Comando SE

Tópicos de Ajuda >> Informaçãos de Uso >> Comandos Básicos >> Comando SE

O que é?

  O comando SE permite que determinadas instruções sejam executadas, ou não dependendo do resultado de ums condição, Ou seja, o algoritmo vai ter mais de uma saída, de acordo com o teste realizado com a condição apresentada.

Como usar?

  Para utilizar o comando SE você deve escrever a palavra reservada SE e entre parênteses colocar a condição, e determinar uma instrução para o teste. Caso saja necessário determinar mais de uma instrução torna-se obrigatório o uso das chaves. Veja a sintaxe na tabela:

COMANDO SE
Desvio Condicional Simples Desvio Condicional Composto Desvio Condicional Aninhado

   se(<condição>)
   {
     <instruções>
   }

ou

 se(<condição>)
    <instrução>

  se (<condição>)
  {
    <instrução_um>
  }senao{
    <instrução_dois>
  }
   se (<condição>)
  {
     <instrução>
  }
    senao
    se (<condição>)
    {
       <instrução>
    }senao{
       <instrução>
    }

Veja os exemplo:

Exemplo 1:

  programa
  {
      funcao inicio ( )
      {
        inteiro num

        escreva ("Digite um número")
        leia (num)
           se (num > 0)
           {
               escreva ("O número é positivo")
           }
      }
  }

Exemplo 2:

  programa
  {
     funao inicio ( )
     {
        se(num > 0)
        {
          escreva ("Positivo")
        }senao{
          escreva ("Negativo")
        }
     }
  }

Exemplo 3:

  programa
  {
       funcao inicio( )
       {
           inteiro num

           escreva ("Digite um numero positivo\n")
           leia (num)

           se(num > 0)
           {
                se(num % 2==0)
                {
                  escreva ("O número: ",num, " é par \n")
                }senao{
                    escreva ("O número: ",num, " é ímpar \n")
                   }senao{
                     escreva ("O numero tem que ser positivo!!")
                }
           }
       }
  }
