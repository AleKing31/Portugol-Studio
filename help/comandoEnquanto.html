Comando enquanto

Tõpicos de Ajuda >> Informações de Uso >> Comandos Bãsicos >> Comando enquanto

O que é?

  O comando ENQUANTO permite que uma sequência de instruções seja executada várias vezes, até que uma condição seja satisfeita, ou seja, repete um conjunto de instruções sem que seja necessario escreve-la várias vezes.

Como usar?

  Para utilizar o comando ENQUANTO você deve usar este comando e entre parênteses colocar a condição, após o fecha parênteses deve se abrir e fechar chaves e entre elas colocar a instrução a ser repetida enquanto a condição for verdadeira. Veja na tabela como utilizar.

COMANDO ENQUANTO

    enquanto(<condição>)
  {
     <instruções>
  }

Veja os exemplo:

Exemplo 1:

  programa
  {
     funcao inicio()
     {
       inteiro x

       x = 1
       enquanto (x < = 5){
         x = x + 1
         escreva (x)
       }
     }
  }

Exemplo 2:

  programa
  {
        funcao inicio()
        {
            inteiro cont = 0

            enquanto (cont < 20)
            {
                se (cont % 2 == 0)
                {
                  escreva ("É um número par ", cont, "\n")
                }
                cont++
            }
        }
  }
