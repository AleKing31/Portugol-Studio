Comando para

Tópicos de Ajuda >> Informaçãos de Uso >> Comandos Básicos >> Comanto para

O que é?

  O comando PARA possui uma variavel de controle, ou seja com está estrutura é possivel executar um conjunto de instruções um número determinado de vezes. Através da variavel de controle, define-se a quantidade de repetições que o laço fará.

Como usar?

  Para utilizar o comando PARA você deverá escrever este comando e entre parêntese inicializar a variavel de controle com determinado número para definir o número de vezes que a intstrução será executada. Veja na tabela como utilizar.

COMANDO PARA

    para(variavel = <valor_inicial>; variavel = <valor_final>; variavel <incremento>)
  {
     <instrução>
  }

Veja os exemplos:

Exemplo 1:

  programa
  {
        funcao inicio( )
        {
            inteiro cont

            escreva ("Digite um número\n")
            para (cont=1; cont<=100; cont++)
            {
               se(cont % 10 == 0)
               {
                  escreva (cont, " é multiplo de 10 \n")
               }
            }
        }
  }

Exemplo 2:

  programa
  {
        funcao inicio( )
        {
           inteiro cont = 1,i

           para (i=1; i<=20; i++)
           {
                 se (cont % 2 == 0)
                 {
                   escreva ("O número ", cont, " é par \n")
                 }
                   escreva ("O número ", cont, " é ímpar \n")

             cont++
           }
        }
  }
