Tipo Real

Tópicos de Ajuda >> Informaçãos de Uso >> Dados e Variaveis >> Tipo Real

O que é?

  Pode-se pensar em um número real como uma fração decimal possivelmente infinita, como 3,14592(...).

Como usar?

  O tipo deve ser declarado juntamente com a variavel. Veja na tabela como utilizar:

TIPO REAL
Formas de Utilizar Exemplo Descrição
1 real numero tipo real com a variavel numero
2 real peso, altura tipo real para usuario atribuir nas variaveis peso e altura
3 real metade, valor metade = valor / 2. A variavel metade do tipo real vai receber o resultado da expressão.

Veja o exemplo completo:

Exemplo 1:

  programa
  {
       funcao inicio( )
       {
          real metade, valor

          escreva ("digite um valor \n ")
          leia (valor)

          metade = valor / 2
          escreva ("A metade do número é: ",metade,"\n")
       }
  }
