Laços de Repetição



Nesta sessão encontram-se exemplos dos comandos "para", "enquanto" e "para-enquanto", usados para fazer determinada ação num numero determinado de vezes ou até uma condição ser satisfeita:
    Exemplos
    • Exemplo 1 (Para)
    • Exemplo 2 (Para)
    • Exemplo 3 (Enquanto)
    • Exemplo 4 (Faça-Enquanto)


Exemplo 1

Neste exemplo, o algoritmo faz uma contagem até o numero digitado pelo usuario


programa

{

funcao inicio (cadeia parametros[])

{

inteiro a,i

escreva ("Digite um numero")

leia (a)

escreva ("\n")

para (i = 0; i <= a; i++) {

escreva (i,"\n")

}

}

}


Exemplo 2

Este exemplo mostra o preenchimento de uma matriz com numeros digitados pelo usuario e a imprime


programa

{

funcao inicio (cadeia parametros[])

{

inteiro a[2][2], i, j

para (i=0;i < 2;i++) {

para (j=0;j<2;j++) {

escreva ("Digite o valor para a posição ",i,"x",j," da matriz ")

leia (a[i][j])

escreva ("\n")

}

}

escreva ("\nMatriz digitada\n")

para (i=0;i < 2;i++) {

para (j=0;j<2;j++) {

escreva (a[i][j]," ")

}

escreva ("\n")

}

}

}



Exemplo 3

Este Exemplo Repete o Exemplo 1, só que com "Enquanto"


programa

{

funcao inicio (cadeia parametros[])

{

inteiro a,i

escreva ("Digite um numero")

leia (a)

enquanto (i <= a) {

escreva (i,"\n")

i++

}

}

}



Exemplo 4

Este Algoritmo roda um contador de 1 até 30 mostrando os numero impares com o laço "Faça-Enquanto"


programa

{

funcao inicio (cadeia parametros[])

{

inteiro a = 1

faca {

se (a % 2 != 0) {

escreva (a," é impar\n")

}

a++

} enquanto (a <= 30)

}

}
