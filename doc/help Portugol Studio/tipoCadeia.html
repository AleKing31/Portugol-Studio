Tipo Cadeia

Tópicos de Ajuda >> Informaçãos de Uso >> Dados e Variaveis >> Tipo Cadeia

O que é?

  O tipo cadeia é um conjunto de caracters

Como usar?

  O tipo cadeia deve estar acompanhado de aspas duplas. Veja na tabela como utilizar:

TIPO CARACTER
Formas de Utilizar Exemplo Descrição
1 cadeia nome tipo cadeia com a variavel nome
2 cadeia endereco, empresa tipo cadeia para usuario atribuir uma letra nas variaveis

Veja o exemplo completo:

Exemplo 1:

  programa
  {
    funcao inicio( )
    {
      cadeia nome

      cadeia = "Alice"
    }
  }

Exemplo 2:

  programa
  {
       funcao inicio( )
       {
          cadeia nome

          escreva ("Digite o teu nome\n")
          leia (nome)

          escreva (nome,"\n")
       }
  }
