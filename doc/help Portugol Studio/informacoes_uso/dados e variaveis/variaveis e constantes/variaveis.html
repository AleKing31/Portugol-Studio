O que é?

  As variaveis servem para armazenar os dados dos nossos programas.

Como usar?

  Todas as variaveis que forem usadas tem que ser declaradas. Veja na tabela como utilizar.

Variaveis
Formas de Utilizar Exemplo Descrição
1 inteiro idade, numero As variaveis do tipo inteiro
2 real media, salario, desconto Variaveis do tipo lógico
3 caracter sexo, letra_inicial As variaveis do tipo caracter.
4 logico recebido A variavel do tipo logico

Veja o exemplo

Exemplo 1:

  programa
  {
       funcao inicio ( )
       {
           inteiro num
           real metade

           escreva ("Digite um número inteiro \n")
           leia (num)
           metade = num /2.0
           escreva ("A metade do número digitado é ", metade)
       }
  }
