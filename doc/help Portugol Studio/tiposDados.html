Tipos de Dados

Tipos Básicos

  Inteiro
  Real
  Caracter
  Cadeia
  Lógico

  Obs: Os números reais usam o ponto como indicador decimal; O tipo caracter deve aparecer entre aspas simples; o tipo cadeia deve ser acompanhado de aspas duplas; e o tipo lógico aceita apenas dois valores (verdadeiro e falso).

Sintaxe:

   < tipo > < variáveis ou lista_de_variaveis >

Exemplo 1:

  programa
  {
    funcao inicio( )
    {
      inteiro num
      real valor
      cadeia nome
      caracter sexo
      lógico opcao

      valor = 6.0
      num = 10
      cadeia = "Giordana"
      sexo = 'f'
      logico = verdadeiro
    }
  }

Exemplo 2:

  programa
  {
       funcao inicio( )
       {
          inteiro num1
          real num2
          caracter letra
          logico vf
          cadeia nome

          escreva ("Digite um número inteiro\n")
          leia (num1)
          escreva ("Digite um número real\n")
          leia (num2)
          escreva ("Digite uma letra\n")
          leia (letra)
          escreva ("Digite o teu nome\n")
          leia (nome)
          escreva ("Digite um valor lógico\n")
          leia(vf)

          limpa()

          escreva (num1,"\n")
          escreva (num2,"\n")
          escreva (letra,"\n")
          escreva (nome,"\n")
          escreva (vf,"\n")
       }
  }
