Comando Leia

Tópicos de Ajuda >> Informações de Uso >> Comandos Básicos >> Comando leia

O que é?

  O Comando LEIA é utilizado quando se deseja obter informações do teclado do computador, ou seja, é um comando de entrada de dados. Esse comando aguarda um valor a ser digitado e o atribui diretamente na variável .

Como usar?

  Para utilizar o comando LEIA, você deverá escrever este comando e entre parênteses colocar a(s) variavel (eis) que você quer que recebam os valores a serem digitados. Como utilizar o comando leia

COMANDO LEIA
Formas de Utilizar Exemplo DescriçÃ£o Resultado
1 leia (a) A variavel a recebe o valor digitado.
2 leia (a,b,c,d,f) As variaveis a,b,c,d,f receberam os valores a ser digitado.
3 leia (valor) A variavel valor receberá um valor a ser digitado pelo usuário.

Veja os exemplos a seguir:

Exemplo 1:

  programa
  {
      funcao inicio ( )
      {
        inteiro numero

        leia (numero)
      }
  }

Exemplo 2:

  programa
  {
      funcao inicio ( )
      {
         real num1, num2, multiplicacao

         leia (num1)
         leia (num2)
         multiplicacao = num1 * num2
      }
  }
