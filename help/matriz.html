Matriz

Tópicos de Ajuda >> Informações de Uso >> Comandos Básicos >> Matriz

O que é?

  As matrizes são variáveis com mais de uma dimensão, normalmente duas, cada posição é referenciada por dois índices uma para linha e outra para coluna, na declaração de matriz indica-se o número de linhas e de colunas destas, e devem aparecer entre conchetes.

Como usar?

  Veja na tabela como declarar matrizes e como utiliza-las.

VETOR
Formas de Utilizar Exemplo Descrição
1 < tipo > matNome [linha][coluna] Sintaxe
2 inteiro matValores[5][5] = {1,2,3} Declaração de matriz com inicialização
3 const inteiro matValores[6][5] = {1,2,3,4,5,6} Declaração de matriz com constante.

Veja o exemplo:

  programa
  {
      funcao inicio( )
      {
        inteiro mat1[3][2]

          mat1[0][0] = 10
          mat1[1][0] = 20
          mat1[2][0] = 30
          mat1[0][1] = 40
          mat1[1][1] = 50
          mat1[2][1] = 60
      }
  }
