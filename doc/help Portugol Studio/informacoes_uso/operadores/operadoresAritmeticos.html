O que é?

  As operações aritméticas são os operadores matemáticos, e são muito usados em algoritmos para realizar cálculos. Veja abaixo quais são estes operadores aritméticos.

Operadores Aritméticos

  + (soma)
  - (subtração)
  / (divisão)
  % (resto divisão)
  * (multiplicação)
