SISTEMA DE AJUDA DO PORTUGOL STUDIO

O que é o Portugol Studio ?

  O Portugol Studio é uma ferramenta para ajudar no desenvolvimento de algoritmos em Português estruturado. A ferramenta auxiliara alunos e professores na aprendizagem de lógica e algoritmos permintindo, os mesmos a construirem seus primeiros algorimos. Para saber mais sobre este software acesse os tópicos de ajuda!
