Atribuição

Tópicos de Ajuda >> Informações de Uso >> Dados e Variaveis >> Atribuição

O que é?

  A atribuição permite modificar o valor de uma variável.

Como usar?

 Para atribuir um valor a uma variável usa-se o sinal de = (igual). Veja na tabela como utilizar o sinal de atribuição

ATRIBUIÇÃO
Formas de Utilizar Exemplo Descrição
1 numero = "15" A variavel numero recebe o valor 15
2 nome = "Portugol" A variavel nome recebe a uma cadeia de caracteres
3 soma = num1 + num2 A variavel soma recebe o resultado da soma das variaveis num1 e num2

Veja os exemplos

Exemplo 1:

  programa
  {
      funcao inicio ( )
      {
         inteiro idade
         cadeia nome

         nome = "Giordana"
         idade = 27
      }
  }

Exemplo 2:

  programa
  {
      funcao inicio ( )
      {
         inteiro a,b,c,d,h
         real f,g

         a = 4
         c = 2
         d = 10
         f = 7.0
         g = 8.0
         h = 80

         g = d / c + 1
         a = a + h
         b = 10
         a = h / b
         f = b * f + g

         escreva ("g ", g, "\n")
         escreva ("a ", a, "\n")
         escreva ("b ", b, "\n")
         escreva ("f ", f, "\n")
         escreva ("h ", h, "\n")
      }
  }
