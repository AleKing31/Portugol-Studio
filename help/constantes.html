Constantes

Tópicos de Ajuda >> Informações de Uso >> Dados e Variaveis >> Constantes

O que é?

  As Constantes são valores que não se alteram durante a execução do programa

Como usar?

  Veja na sintaxe e nos exemplos como usar as constantes

Sintaxe:

   const < tipo > < nome_variavel > = < valor >

Exemplo 1:

  programa
  {
    funcao inicio( )
    {
      const inteiro num = 10
      const real valor = 6.0

      :
      :
    }
  }

Exemplo 2:

  programa
  {
       funcao inicio( )
       {
          const inteiro anoDeNascimento = 1983
          const inteiro anoAtual = 2010
          inteiro idade

          idade = anoAtual - anoDeNascimento
          escreva ("A idade é ", idade)
       }
  }
